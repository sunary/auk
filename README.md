# Elixir pipeline

Elixir pipeline using genstage

## Structure

```
  .
  +-- components/
  +-- assemblies/
  |   +-- job1/
  |       +-- app_config.exs
  |   +-- job2/
  |       +-- app_config.exs
  |   +-- job3/
  |       +-- app_config.exs
  +-- functions/
  +-- test/
  +-- mix.exs
```

## Installation

```sh
$ mix deps.get
```

## Run

```sh
$ iex -S mix
```
